# 🔖Today I Learned
毎日の学習内容を記録するリポジトリです。

## 🎯 目的
- 学習を継続的に行う
- アウトプットで理解を深める
- 日々の成長を記録として残す

## 🚀 目標
- 1日1コミットを目指す
- 1ヵ月で学んだトピックからピックアップして、まとめ記事を1本作成する

## 📝 運用ルール
- 行った内容を、[DailyReport](./00_DailyReport)に毎日記録する
  - 内容は技術的なことに限らず、業務で学んだことや気づきも含む
- 知識の整理やアウトプットを記録する
  - 各トピックごとにフォルダを作成し、関連する学習内容をまとめる
- 1ヵ月ごとに振り返りを行う
  - 来月の学習計画を立てる
